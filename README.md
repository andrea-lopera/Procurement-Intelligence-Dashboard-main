# Production-Line-Intelligence-Dashboard
Simulated production analytics dashboard using Python, SQL, and Power BI.
## Project Summary 
A simulated manufacturing analytics project to track KPIs, downtime, and rework for a tempered glass door production line.

## Goal
- Build a realistic dashboard using Python, SQL, and Power BI
- Highlight root cause analysis, KPI tracking, and downtime reduction
- Showcase operational data science skills

---

## Project Structure

1. **Step 1: Designing the Production Process**  
   → Stations: Cutting, Tempering, Framing, QC  
   → Data columns: unit_id, shift, rework_flag...

2. **Step 2: Data Simulation in Python**  
3. **Step 3: Exploratory Analysis (Python)**  
4. **Step 4: SQL Reporting + KPI Queries**  
5. **Step 5: Power BI Dashboard**  
6. **Step 6: Insights + Recommendations**

---

## Tools Used
- Python (pandas, seaborn)
- SQL (SQLite)
- Power BI

## Key Metrics
- Rework Rate
- Yield
- Avg. Downtime
